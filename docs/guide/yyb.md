# 应用宝配置

## 配置说明

应用宝是腾讯旗下的安卓应用分发平台，需要配置`user_id`和`access_secret`。

::: warning 应用宝特殊配置
需要注意的是应用宝需要在应用配置的时候配置一个特殊的应用宝appId
:::

## 配置项

```json
{
  "user_id": "xxx",
  "access_secret": "xxx"
}
```

### 参数说明

<img src = "/images/config_yyb.png" width="25%" />

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| user_id | string | 是 | 用户ID |
| access_secret | string | 是 | 应用Key |
