# vivo 应用商店配置

## 配置说明

vivo 应用商店是 vivo 官方的应用分发平台，需要配置`access_key`和`access_secret`。

## 配置项

```json
{
  "access_key": "xxx",
  "access_secret": "xxx"
}
```

### 参数说明

<img src = "/images/config_vivo.png" width="25%" />

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| access_key | string | 是 | API访问密钥Key |
| access_secret | string | 是 | API访问密钥Secret |
