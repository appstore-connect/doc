# 荣耀软件商店配置

## 配置说明

荣耀软件商店是 荣耀官方的应用分发平台，需要配置`client_id`、`client_secret`。

## 配置项

```json
{
  "client_id": "xxx",
  "client_secret": "xxx"
}
```

### 参数说明

<img src = "/images/config_honor.png" width="25%" />

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| client_id | string | 是 | 客户端ID |
| client_secret | string | 是 | 客户端密钥 |
